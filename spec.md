# Dr. Rajneesh Gupta Clinic Website

## Overview
A professional, minimal, and responsive website for Dr. Rajneesh Gupta Clinic inspired by medantaeclinic.org design style, featuring clean design with calm color palette (white, soft blues, and greens), sticky navigation header, smooth scroll navigation, subtle animations, and SEO optimization.

## Navigation Structure
Sticky header with clinic name "Dr. Rajneesh Gupta Clinic" and navigation menu:
- Home
- About Us  
- Reviews
- Contact Us
- Appointment

All navigation links (both header and footer) must reset scroll position to the top of the page when navigating between routes, ensuring consistent user experience with smooth scroll transitions.

## Design & UX Requirements
- Clean minimalist text-only design with light color background blocks replacing all image placeholders
- Large headings with balanced whitespace and intuitive flow
- Clean icons and responsive layout across all screen sizes
- Soft animations on scroll and smooth transitions
- Fully responsive design optimized for mobile, tablet, and desktop
- Clean, professional medical website style with ample white space, clear typography, and well-separated sections
- Minimal typography consistent with clinic aesthetic with balanced alignment and spacing for high readability
- Ensure proper layout alignment and consistent spacing throughout all sections
- Maintain visual balance and proper spacing when displaying reduced list items in the "Why Choose Us" section
- Ensure no layout gaps or visual imbalances after element removal; maintain clean alignment and responsive spacing
- Remove the phrases "valid id proof" and "insurance information" from specific list items while maintaining all other text, formatting, spacing, and layout exactly as they are
- Keep layout responsive and maintain spacing consistency with surrounding content
- No image files referenced in any frontend pages - complete image-free design
- **Navigation Scroll Behavior:** All internal route navigations must reset scroll position to the top of the page, ensuring consistent user experience across header and footer navigation links
